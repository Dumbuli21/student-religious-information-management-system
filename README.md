## About SRIMS

The Student Religious Information System (SRIS) is a web-based application designed to manage religious-related information for students within a university or college environment. The system
provides a centralized platform where different religious groups can operate independently within a single system.
Each religion can manage its own students, announcements, and events without interfering with other religions. The system promotes inclusivity, organization, and effective communication
among students and religious organizations within the institution.

2.3 User Classes and Characteristics
2.3.1 Super Administrator
•Has full access to the system
•Manages religions and system settings
•Requires basic computer literacy2.3.2 Religious Administrator
•Manages information for a specific religion
•Can create and manage Sub-Administrators within the same religion
•Oversees student registration, announcements, and events
•Views reports and attendance
2.3.3 Sub-Administrator (Religion-Level)
•Assigned by the Religious Administrator
•Belongs to a specific religion only
•Assists in registering and managing students
•Manages news, announcements, and events for the assigned religion
•Cannot access data of other religions
2.3.4 Student
•Views information related to their religion
•Registers for events
•Submits feedback
