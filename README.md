## 📌 About SRIMS

The **Student Religious Information Management System (SRIMS)** is a web-based application designed to manage religious-related information for students within a university or college environment.

The system provides a centralized platform where different religious groups can operate independently within a single system. Each religion can manage its own students, announcements, and events without interfering with other religions.

The system promotes **inclusivity, organization, and effective communication** among students and religious organizations within the institution.

---

## 👥 User Roles

| Role | Description |
|------|-------------|
| **Super Admin** | Has full access to the system. Manages religions and system settings. |
| **Religious Admin** | Manages information for a specific religion. Can create and manage Sub-Administrators within the same religion. Oversees student registration, announcements, and events. Views reports and attendance. |
| **Sub Admin** | Assigned by the Religious Administrator. Belongs to a specific religion only. Assists in registering and managing students. Manages news, announcements, and events for the assigned religion. Cannot access data of other religions. |
| **Student (User)** | Views information related to their religion. Registers for events. Submits feedback. |
